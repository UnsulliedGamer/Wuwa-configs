# Wuwa-configs
Wuthering Waves (WuWa) Mobile Config Tweaks

This repository contains custom configuration (.ini) tweaks for Wuthering Waves (WuWa) aimed at improving visual clarity, reducing graininess, and balancing performance on mobile devices.

These configs are especially useful for players experiencing:

- Blurry or grainy visuals
- Oversharpening artifacts
- Inconsistent resolution scaling
- FPS drops caused by unnecessary effects
